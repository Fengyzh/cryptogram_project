<------ Please do the following in order ------>


Database
-------------------
1. Open bash terminal
2. run "bash setupDB.sh" or "./setupDB.sh"


Server
-------------------
1. At root directory
2. node server.js

** Server Listens on port 4000 **


Client
-------------------
1. cd client
2. npm start (Might take a while)

** Client Listens on port 3000 **
